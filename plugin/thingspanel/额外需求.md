# MQTT主题数据转换需求

## 功能描述
当设备发来主题为 `custom/up/+` 时，将数据转发给 `telemetry/gateway`，按照网关规范转换payload格式。

- 原主题的 `+` 代表子设备地址
- 保持原始数据结构不变，仅进行格式包装

## 转换示例

**输入:**
- 主题: `custom/up/device001`
- Payload:
```json
{
    "temperature": 28.5,
    "version": "v0.1",
    "switch": true
}
```

**输出:**
- 主题: `telemetry/gateway`  
- Payload:
```json
{
    "sub_device_data": {
        "device001": {
            "temperature": 28.5,
            "version": "v0.1",
            "switch": true
        }
    }
}
```
